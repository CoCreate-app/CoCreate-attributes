CoCreate-domEditorPanel

Simple HTML5 & JavaScript component operate add, update & remove values in element's attributes from input, select or js api. Easy configuration using class, style or any custom attribute. Highly customizable and styleable.

Install

 
npm i @cocreate/action

Or you can use cdn link: 

<script>https://cdn.cocreate.app/attributes/latest/CoCreate-attributes.min.js</script>
